# myportfolio

Mejoras implementadas con respecto al PF02
1. Validación de formulario con mensajes de error. Al enviar el formulario se muestra una confirmacion, en caso contrario, se mostrara un mensaje de error
2. Modo Oscuro/Claro con guardado en LocalStorage. Se incorporo un boton tematico de Sol/Luna para cambiar entre modo oscuro y claro en el portfolio
3. Pop-up de confirmación de envío del formulario. Al enviar el formulario se mostrara un mensaje de confirmacion.
4. Contador de visitas con LocalStorage. Ahora hay un contador de visitas hecho en JavaScript, donde cada visita o recarga de la pagina sera contabilizada en la columna izquierda y guardada en el dispositivo.
5. Animaciones dinámicas con JavaScript. Ahora se incorporaron animaciones en el navbar con JavaScript. Especificamente al posicionar el cursor sobre alguna de las opciones, esta aumentara su tamaño y cambiara de color a traves de una animacion.
